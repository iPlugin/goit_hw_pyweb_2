# Poetry - це інструмент для управління залежностями у Python проектах 

# Створення нового проекту
# poetry new <ім'я проекту> - створення "poetry-проекти"
# Розділ tool.poetry - призначений для опису проекту: назва, версія, коротка інформація
# tool.poetry.dependencies - тут вказані всі production залежності
# tool.poetry.dev-dependencies - призначений для залежностей, які використовуються під час розробки

# Ініціалізація вже існуючого проекту
# poetry init - щоб ініціалізувати poetry, у вже готовому проекті
# poetry shell - для активації віртуального середовища

# Основні команди
# poetry add aiosqlite - Щоб додати до проекту залежність у вигляді aiosqlite
# poetry add pytest --dev - Щоб додати залежність для розробки --dev
# poetry remove aiosqlite - щоб видалити залежність
# poetry install - якщо потрібно підтягнути існуючий проект і встановити його залежність
# poetry update - оновити всі залежності до останніх версій
# poetry show - щоб переглянути всі доступні пакети
